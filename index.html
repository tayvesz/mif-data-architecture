Enhancing the IIAG with Geocoded Conflict Data

Strategic Objective:

To enhance the IIAG‚Äôs ‚ÄúSecurity & Rule of Law‚Äù category with near real-time, geolocated conflict data, enabling dynamic governance insights.

Proposed Implementation:

  1. Automated Data Ingestion: Develop containerized Cloud Functions (e.g., Google Cloud Functions) to pull daily event data from the ACLED API, ensuring robust validation and storage in a Data Lake.
  2. Data Transformation: Leverage DBT (Data Build Tool) to create version-controlled SQL models in BigQuery, embedding automated quality checks for reliable metrics.
  3. Visualization: Enable researchers to use Power BI or Tableau for interactive maps and time-series analyses of conflict hotspots.

Gauging Public Discourse with Multilingual NLP

Strategic Objective:

To capture real-time public sentiment on policy issues through multilingual Natural Language Processing (NLP), strengthening the IIAG‚Äôs ‚ÄúParticipation, Rights & Inclusion‚Äù category.

Proposed Implementation:

  1. Data Collection: Deploy Python scripts on Cloud Run to scrape articles from key African news outlets using targeted keywords.
  2. Ethical AI Analysis: Combine Google‚Äôs Natural Language API with Hugging Face models for sentiment and entity recognition in African languages such as Swahili and Hausa.
  3. Insights Delivery: Store results in BigQuery and develop dashboards to track sentiment trends transparently, noting model limitations.

Monitoring Infrastructure with Satellite Imagery

Strategic Objective:

To provide evidence-based monitoring of public infrastructure projects, enhancing accountability and governance transparency.

Proposed Implementation:

  1. Data Acquisition: Source open satellite imagery from Sentinel-2, ensuring cost-effective and accessible data inputs.
  2. AI Prototyping: Use Vertex AI AutoML Vision to train image recognition models for detecting infrastructure changes, enabling rapid deployment.
  3. Automation: Develop Python scripts to automate imagery processing and store results in BigQuery for geospatial visualization.
